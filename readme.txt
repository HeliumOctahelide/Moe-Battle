NGA舰萌计票辅助系统v2 by 江天万里霜

特性：
* 完全封装了整个计票过程，从打开网页到生成计票结果文件都在一个程序内实现。
* 自带自动识别功能，可以自动完成80%以上的工作，人工只需要查漏补缺。
* 纯键盘操作，计票速度极快。
* 支持保存进度，以及标记特定的楼层。
* 因为只花了一晚上来写，天知道会有什么bug。

使用方法：
运行程序，写入候选人名单，并读取投票专楼中你要计票的楼层，然后逐楼进行计票。
回复内容会以输入的候选人名单自动识别选票，只要输入的候选人名单里包含了常见的别名，在大部分情况下就都可以完全识别。

moewar.exe是封装好的可执行文件，运行这个就可以了；Win10下可用，别的环境就有请沙雕群友们测试了。

moewar.py是源代码。zh_wiki.py和langconv.py是实现简繁转换的两个包。

config.ini是配置和存档文件。

vote.json是存放帖子内容的文件。

计票结果会输出为result.csv。

压缩包中已经带有一个示例，它是对2018年萌战小组赛A组的第12到13页的计票结果。

有什么不懂的或者要汇报bug直接群内@就好。
